# Insurance Portfolio Risk Analysis – BI Intern Assignment - Zopper

## Overview
This project analyzes an insurance portfolio to evaluate risk exposure, claim trends, and overall profitability. Synthetic datasets were generated to simulate policy sales and insurance claims. SQL was used for analytical queries, and a dashboard was created to visualize key performance indicators.

## Project Objectives

- Analyze insurance policy and claims data.
- Calculate key performance metrics such as loss ratio and claim frequency.
- Identify risk patterns in the insurance portfolio.
- Build a dashboard for visualizing trends and insights.

## Tools Used

- Python (for dataset generation)
- Jupyter Notebook
- MySQL Workbench (for database management and SQL analysis)
- Power BI (for dashboard visualization)

1. Database_Creation.ipynb

9. Dashboard Image.png


## Key Metrics Analyzed

- Total Policies Sold
- Total Claims Filed
- Portfolio Loss Ratio
- Monthly Claim Trends
- Average Claim Cost by Vehicle Type

## Key Insights

- The calculated portfolio loss ratio indicates the relationship between claim payouts and premium collected.
- Monthly claim analysis highlights claim distribution across the year.
- Certain vehicle categories show higher claim costs, indicating potential high-risk segments.

## Business Recommendations

- Adjust premium pricing for high-risk vehicle segments.
- Implement risk-based underwriting strategies.
- Use predictive analytics to monitor future claim trends.
